# Repositorio Arminiano

Sitio web estático de teología arminiana clásica. Desplegado en GitHub Pages.

## Estructura

```
repositorio-arminiano/
├── index.html                          ← Sitio completo (auto-contenido)
├── README.md                           ← Este archivo
├── 00_INDICE.md                        ← Índice general
├── 01_la_gracia_preveniente.md
├── 02_la_eleccion_condicional.md
├── 03_la_expiacion_ilimitada.md
├── 04_depravacion_total_y_respuesta_humana.md
├── 05_la_doble_predestinacion_critica_arminiana.md
├── 06_agustin_maniqueismo_y_determinismo.md
├── 07_libre_albedrio_y_responsabilidad_moral.md
└── 08_posibilidad_de_caer_de_la_gracia.md
```

## Agregar nuevos artículos

1. Creá el archivo `.md` con el formato estándar (ver artículos existentes)
2. Subilo a la carpeta **Repositorio arminiano** en Google Drive
3. Agregá la referencia en `index.html` dentro del array `ARTICLES`

## Desplegar en GitHub Pages

1. Creá un repositorio en GitHub
2. Subí todos los archivos
3. En Settings → Pages → Source: `main / root`
4. El sitio quedará disponible en `https://[usuario].github.io/[repo]`
